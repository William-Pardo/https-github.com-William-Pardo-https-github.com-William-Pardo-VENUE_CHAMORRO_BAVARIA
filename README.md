# Walkthrough 360 — Bavaria

Visor 360 Pannellum offline/local y publicado en GitHub Pages.

## Uso local

1. Abrir `INICIAR_SERVIDOR_LOCAL.bat`.
2. Entrar a `http://localhost:8000/index.html`.
3. Navegar usando los puntos interactivos.

## Estructura

- `index.html` — walkthrough principal.
- `canva.html` — versión puente limpia para insertar en contenedores externos.
- `pannellum/` — librería local del visor.
- `Render/BAVARIA/` — panoramas.
